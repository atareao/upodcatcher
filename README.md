Download
--------

Download the [package][dlp] *or* checkout the source:

    $ git clone https://github.com/atareao/upodcatcher.git

Installation
------------

    $ sudo add-apt-repository ppa:atareao/upodcatcher
    $ sudo apt update
    $ sudo apt install upodcatcher
